# Changelog

All notable changes to SkyBox will be documented in this file.

## [1.0.0] - 2026-01-23

### Added

- Initial release of SkyBox
- Telegram-themed file explorer with dark glass UI
- File browsing capabilities (list, open, create, rename, delete)
- Directory navigation with breadcrumbs
- Search functionality
- Favorites management
- Recent files tracking
- Session persistence with SQLite database
- Keyboard shortcuts (refresh, delete, escape)
- Responsive design for various screen sizes

### Features

- Secure filesystem operations with path validation
- Cross-platform desktop application using Tauri
- Offline-first architecture
- Modern React + TypeScript frontend
- Rust-based backend for system operations
- Local SQLite database for user data

## [0.1.0] - 2025-12-XX

### Added

- Project scaffolding with Tauri, React, TypeScript, and Tailwind CSS
- Basic file explorer UI with Telegram-inspired design
- Login screen with phone and QR code options
- Initial filesystem commands implementation
- Database module with user session support
